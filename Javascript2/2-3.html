각 회의 승자 출력, 비겼을 경우 D 출력

            A : 1  1  1  2  2  2  3  3  3
            B : 1  2  3  1  2  3  1  2  3  (-)
        -------------------------------------------
            =   0 -1 -2  1  0  -1 2  1  0

            A - B == 0  : D (비긴거)
            A - B == -1 : B 승
            A - B == -2 : A 승
            A - B == 1  : A 승
            A - B == 2  : B 승
